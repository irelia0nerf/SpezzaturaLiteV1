# F2F-RAaT Glossary

- **F2F-RAaT**: From Fact to Feedback, a framework for Reputation-as-a-Transaction.
- **Spezzatura Engine (TÂ²)**: The deterministic reputational engine based on hybrid vectors.
- **Sigmoid P(x) Score**: A reactivity layer for handling acute, time-sensitive events.

- **Burn Engine**: The deterministic executor of reputation-based policies.
- **Veritas Protocol**: The protocol for sealing, hash-chaining, and proving the integrity of decisions.

- **Policy**: A versioned, signed set of authorized rules and effects.

- **Bounded-Output Rationale**: A sealed explanation with controlled entropy, containing no PII.
